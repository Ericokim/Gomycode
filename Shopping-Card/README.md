# Shopping Card

A fully functional shopping cart application with interactive features built using vanilla JavaScript.

## Features

- Add and adjust item quantities with plus and minus buttons
- Remove items from cart with delete functionality
- Like items with a clickable heart button that changes color
- Automatic total price calculation based on quantities
- Responsive design with Bootstrap
- Clean and intuitive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- Font Awesome Icons

## Functionality

The shopping cart includes the following interactive features:

1. **Quantity Control**: Use the plus and minus buttons to increase or decrease item quantities. The minus button won't reduce quantity below zero.

2. **Delete Items**: Click the trash icon to remove items completely from the cart.

3. **Like Items**: Click the heart icon to toggle between liked and unliked states. The heart changes to red when liked.

4. **Total Price**: The total price automatically updates based on item quantities and deletions.
