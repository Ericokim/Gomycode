# React Hooks Checkpoint

This project is a simple movie app built with React hooks.

## Features

- Display favorite movies or TV shows in reusable movie cards
- Filter the list by title
- Filter the list by minimum rating
- Add a new movie with title, description, poster URL, and rating
- Use Tailwind CSS classes grouped in `styles` objects inside components

## Main Components

- `MovieCard`
- `MovieList`
- `Filter`

## Scripts

### `npm start`

Runs the app in development mode.

### `npm test`

Runs the test suite.

### `npm run build`

Builds the app for production.
