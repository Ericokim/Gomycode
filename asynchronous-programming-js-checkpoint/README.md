# Asynchronous Programming in JavaScript

This checkpoint demonstrates simple asynchronous programming patterns in JavaScript using `async`/`await`.

## Covered Tasks

- Task 01: Iterating with `async`/`await`
- Task 02: Awaiting a simulated API call
- Task 03: Handling errors with `async`/`await`
- Task 04: Awaiting concurrent requests
- Task 05: Awaiting parallel calls

## How To Run

From this folder, run:

```bash
node index.js
```

No external packages are required.
