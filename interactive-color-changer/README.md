# Interactive Color Changer

A simple and interactive web application that changes the color of a box with the click of a button.

## Features

- Random color generation using hexadecimal values
- Smooth color transitions
- Clean and modern UI design
- Responsive button with hover effects

## How It Works

Click the "Change Color" button and watch the box change to a random color. The color is generated using a random hexadecimal code.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
