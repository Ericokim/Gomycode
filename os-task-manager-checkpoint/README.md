# Designing a Simple OS Task Manager

## Overview

This checkpoint simulates how an operating system manages multiple tasks and shared resources.

The full submission report is here:

[os-task-manager-report.md](os-task-manager-report.md)

PDF version:

[os-task-manager-report.pdf](os-task-manager-report.pdf)

## Covered Topics

- CPU scheduling with FCFS and Round Robin
- Gantt charts and average waiting time
- Process synchronization with a mutex
- FIFO and LRU page replacement
- FCFS and SSTF disk scheduling

## Key Results

| Topic | Result |
