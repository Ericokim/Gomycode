# Shopping Cart Checkpoint

## Description

This project is a simple **Shopping Cart Web Application** built with **HTML, CSS, and JavaScript**.  
It allows users to manage their shopping cart by updating item quantities, liking items, and removing items. The total price updates dynamically as changes are made.

## Features

- Adjust Quantity: Increase or decrease item quantity using `+` and `-` buttons
- Like Items: Click the heart icon to like/unlike items (heart turns red when liked)
- Remove Items: Click the trash icon to delete items from the cart
- Automatic Total Update\*\*: Total price updates immediately after any action
- Data-Driven Prices: Each item’s price is stored in a `data-price` attribute
- Quantity Validation: Quantity cannot go below 0
