# API Checkpoint Users App

This React app consumes the JSONPlaceholder users API and displays the returned users on screen.

## Built With

- Create React App
- React `useEffect`
- React `useState`
- Axios
- Tailwind CSS

## API

```text
https://jsonplaceholder.typicode.com/users
```

## Main Files

- `src/App.js` - page layout and header
- `src/UserList.js` - axios request, state, loading/error handling, and user rendering
- `src/index.css` - Tailwind setup and base styles

## Run The App

```bash
npm start
```

## Build

```bash
npm run build
```
