# Gomycode Projects

This repository contains various web development projects and exercises completed during the Gomycode bootcamp.

## Projects

### Shopping Cart

A fully functional shopping cart application with interactive features including quantity adjustment, item deletion, like functionality, and automatic total price calculation. Built with vanilla JavaScript, Bootstrap, and Font Awesome icons.

### Portfolio (Tailwind CSS)

Modern and responsive personal portfolio website built with Tailwind CSS, featuring project showcase, skills section, and contact form.

### Blog (Tailwind CSS)

A blog website built using Tailwind CSS for styling and responsive design.

### Delicious Bites Menu

A restaurant menu webpage built with HTML, featuring appetizers, main courses, and desserts.

### Tasty Bites

Restaurant landing page showcasing menu items and services.

### MyStore Landing Page

E-commerce landing page with product displays and modern design.

### Travel Agency Landing Page

Landing page for a travel agency featuring destinations and travel packages.

### Docplanner

Clone of Docplanner website interface with HTML and CSS.

### JavaScript Algorithms

Collection of JavaScript algorithm implementations including:

- Insertion Sort
- String manipulation exercises
- Array operations

### String, Math & Array

JavaScript exercises focused on string methods, mathematical operations, and array manipulations.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- Bootstrap
- Font Awesome

## Getting Started

Each project folder contains its own HTML, CSS, and JavaScript files. Simply open the `index.html` file in a web browser to view the project.
